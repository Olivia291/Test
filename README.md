# 划词翻译 - Chrome 浏览器插件

轻量化划词翻译插件，选中英文单词即可查看中文释义。

## 功能

- ✅ 划词翻译：选中英文单词，自动弹出中文释义
- ✅ 词性显示：自动识别并显示单词词性
- ✅ 英式发音：点击朗读按钮，使用英式发音朗读单词
- ✅ 点击空白关闭：点击页面任意空白区域即可关闭弹窗
- ✅ 轻量简洁：无多余功能，专注翻译

## 安装

1. 下载或克隆本仓库
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本仓库文件夹

## 使用

1. 在任意网页上选中一个英文单词
2. 自动弹出翻译结果弹窗
3. 点击 🔊 按钮收听英式发音
4. 点击页面空白处关闭弹窗

## 技术栈

- Chrome Extension Manifest V3
- 百度翻译 API
- Web Speech API（英式发音）

## 配置

如需更换百度翻译 API 密钥，请修改 `content.js` 中的配置：

```javascript
const APP_ID = "你的APP ID";
const SECRET_KEY = "你的密钥";
```
